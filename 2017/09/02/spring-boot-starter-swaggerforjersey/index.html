自制Jersey-Swagger的spring-boot-starter

发表于 2017-09-02 | 分类于 微服务 | | Views 次

Spring Boot的自动化配置特性来实现快速的将swagger2引入spring boot应用来生成jersey的API文档，简化原生使用swagger2的整合代码。

版本基础

  • Spring Boot：1.5.x
  • swagger-jersey2-jaxrs：2.7.x
  • Jersey 2

如何使用
